**💡 产品方案**：独立设计并构建专属的图文知识库，完成 AI 工具向 C 端问诊产品的闭环转化。

⚙️ 底层技术逻辑：
* 轻量级 RAG 检索：在本地知识库中，基于 TF-IDF 与余弦相似度精准检索相关的症状、防治和用药条目。
* Context 注入机制：将检索结果拼接为上下文注入 Prompt，再调用 OpenAI 兼容大模型生成高准确率的诊断建议，有效控制幻觉并保证来源可追溯。

### 2️⃣ 病虫害 CV 监控及可视化大屏系统
**能力标签**：`YOLOv8` `WebSocket` `Vue3` `数据可视化`

**🎯 业务痛点**：传统农业监控仅有实时画面而无结构化数据沉淀，无法做到全天候的异常干预。
**💡 产品方案**：从业务场景出发，主导视觉模型推理部署与前端监控打通，构建“识别-预警-推送”的完整物联网数据流。

⚙️ 商业监控逻辑：
* 双通道低延迟传输：视频帧流媒体 (MJPEG) 与结构化检测数据 (WebSocket) 分离传输，保障弱网环境下的前端大屏秒级响应。
* 数据驱动分级预警：按 YOLOv8 实时识别的病害数量设定 Normal/Warning/Danger 阈值，并结合温湿度传感器数据辅助多维度的决策干预。
